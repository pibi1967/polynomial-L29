The Record Polynomial

This monic cubic polynomial, discovered by Paolo Borghi, generates 30 consecutive primes:

f(n) = n³ + 23n² − 1988n + 24223

It produces prime values for all integers n from 0 to 29.

Verification Code (Python)

import sympy as sp def f(n): return n**3 + 23*n**2 - 1988*n + 24223 def is_prime(n): return sp.isprime(n) L = 0 for n in range(100): if is_prime(f(n)): L += 1 else: break print("Run length L =", L)

About This Result

This constitutes the longest known run of primes generated by any monic cubic polynomial with integer coefficients. The search domain included:

  • a ∈ [−2000, 2000]
  • b ∈ [−2000, 2000]
  • c prime, 2 ≤ c ≤ 50000
