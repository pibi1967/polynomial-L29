A Structural Prime-Generating Monic Quartic Polynomial

Documented by Paolo Borghi — Verified computationally (2026)

The quartic polynomial (L = 37)

We consider the monic quartic polynomial

f(n) = n⁴ − 41n³ + 550n² − 2622n + 4099

This polynomial produces a run of 37 consecutive prime values when evaluated at consecutive integers starting at n = 0. As usual in this context, primality is tested on the absolute value |f(n)|.

Category: monic quartic prime-generating polynomial

The polynomial

f(n) = n⁴ − 41n³ + 550n² − 2622n + 4099

is a monic quartic polynomial, since the leading coefficient (the coefficient of n⁴) is equal to 1. This places it within the class of monic quartic prime-generating polynomials, a mathematically significant and more restrictive family compared to non-monic quartics.

In this stricter category, achieving a long consecutive prime run is notably difficult. The quartic above attains a run length of:

L = 37

establishing an exceptionally long documented prime-generating run for a monic quartic polynomial. This record is of a structural nature: it is obtained by a sequence of translations starting from a polynomial with run length L = 33, as explained below.

From L = 33 to L = 37 via translation

The starting polynomial (L = 33)

The base of the genealogy is the polynomial

g(n) = n⁴ − 25n³ + 154n² + 66n + 43

which produces a run of L = 33 consecutive primes starting at n = 0. In other words, |g(n)| is prime for n = 0, 1, …, 32, and the first composite value occurs at n = 33, namely:

g(33) = 457423 = 103 × 4441.

The n → n − 1 translation method

Given a polynomial P(n) with a long prime run, we can consider the translated polynomial

Q(n) = P(n − 1).

If the newly introduced value P(−1) is also prime, then the prime run of Q(n) starting at n = 0 is extended by one:

  • P(n) prime for n = 0, …, L − 1
  • P(L) composite
  • P(−1) also prime

implies that Q(n) = P(n − 1) is prime for n = 0, …, L, with the first composite at n = L + 1. Thus we obtain a new polynomial with run length L + 1.

Iterating this procedure constructs a genealogical tree of quartic polynomials. Each step is a translation by n → n − 1, and the run length increases or decreases accordingly.

The genealogical tree (L = 37 → L = 1)

Starting from the structural record f(n) = n⁴ − 41n³ + 550n² − 2622n + 4099 with L = 37, one obtains a full genealogy of monic quartic polynomials by successive translations. Each polynomial f_L(n) in the tree has the form

f_L(n) = n⁴ + a n³ + b n² + c n + d

and generates exactly L consecutive primes starting at n = 0. The first composite value occurs at n = L, and in this family it is always

f_L(L) = 457423 = 103 × 4441.

Computational verification (sketch)

The structural record and the entire genealogical tree were established by exhaustive evaluation of each quartic and deterministic primality checks up to the first composite value. For all values involved, deterministic methods for 64-bit integers (including trial division up to √x or deterministic Miller–Rabin) are sufficient.

For the structural record f(n) = n⁴ − 41n³ + 550n² − 2622n + 4099 we have:

  • |f(n)| is prime for n = 0, 1, …, 36
  • first composite at n = 37
  • f(37) = 457423 = 103 × 4441

The same composite value 457423 appears at the boundary (at n = L) for every polynomial in the genealogical table, which reflects the way the family is constructed via translations.

Reproducibility: minimal Python snippet

Any reader can verify the run length of the structural record with the following code: 

"""
Deterministic primality test for the structural monic quartic record:

    f(n) = n^4 - 41 n^3 + 550 n^2 - 2622 n + 4099
"""

A = -41
B = 550
C = -2622
D = 4099

def is_prime(n: int) -> bool:
    if n < 2:
        return False
    if n % 2 == 0:
        return n == 2
    d = 3
    while d * d <= n:
        if n % d == 0:
            return False
        d += 2
    return True

def f(n: int) -> int:
    return n**4 + A*n**3 + B*n**2 + C*n + D

def main():
    primes = []
    n = 0
    while True:
        v = f(n)
        if v < 2 or not is_prime(abs(v)):
            break
        primes.append((n, v))
        n += 1

    L = len(primes)
    print("Structural quartic record:")
    print(f"  (a, b, c, d) = ({A}, {B}, {C}, {D})")
    print("\nLength of the consecutive prime run (L):", L)
    print(f"First composite at n = {L}: f(n) = {f(L)}")

if __name__ == "__main__":
    main()

Running this program will output L = 37, confirming the claimed structural record. By adjusting the coefficients to those in the genealogical table, one can similarly verify each descendant polynomial.
